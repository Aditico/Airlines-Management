# Airline Reservation System

This README provides instructions on how to import the provided project zip file into NetBeans, how to import the provided database SQL file into the imported project on NetBeans using Workbench/PhpMyAdmin, and information about the CRC cards, class diagram, and use case diagram provided.

## Importing the Project Zip File into NetBeans

1. Open NetBeans IDE.
2. Go to **File > Open Project**.
3. Navigate to the location of the downloaded zip file.
4. Select the zip file and click **Open**.
5. The project should now be imported and visible in the **Projects** window.

## Importing the Database SQL File

### Using MySQL Workbench

1. Open MySQL Workbench.
2. Connect to your MySQL server.
3. Click on **Server > Data Import**.
4. Choose **Import from Self-Contained File**.
5. Navigate to the location of the SQL file and select it.
6. Click **Start Import**.

### Using PhpMyAdmin

1. Open PhpMyAdmin in your web browser.
2. Select your database on the left panel.
3. Click on **Import** tab.
4. Click **Choose File** and navigate to the SQL file.
5. Click **Go** at the bottom of the page to import.

## CRC Cards, Class Diagram, and Use Case Diagram

The project submission also includes CRC cards, a class diagram, and a use case diagram. 

- **CRC Cards**: These cards provide a simple way to document a class's responsibilities and collaborations. They are an effective tool for conveying design in a form that can be easily understood, yet contains sufficient detail to actually implement the design.

- **Class Diagram**: This diagram shows a set of classes, interfaces, and collaborations and their relationships. It provides a wide range of usages; from modeling the static view of an application to modeling the detailed design.

- **Use Case Diagram**: This diagram presents a graphical overview of the functionality provided by a system in terms of actors, their goals (represented as use cases), and any dependencies between those use cases.

Please refer to these documents for a better understanding of the project's design and functionality.
